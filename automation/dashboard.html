Setup-Anleitung (einmalig, ca. 45 Minuten)

1
Cloudflare Worker deployen
Account anlegen auf cloudflare.com → Workers → Neue Worker → Code aus worker.js einfügen
Umgebungsvariablen setzen: BREVO_API_KEY, SILVIO_EMAIL, SHEETS_WEBHOOK_URL
2
Brevo Account anlegen (kostenlos)
brevo.com → Kostenlos registrieren → API-Key kopieren → in Cloudflare Worker als BREVO_API_KEY eintragen
Absender-E-Mail verifizieren: ga-richter@freenet.de
3
Google Sheet anlegen
Neues Google Sheet: "Richter_Leads" mit Spalten: Datum, Name, Email, Telefon, Thema, Rueckruf, Status, Quelle
Zweites Sheet: "Richter_Kunden" mit: Name, Email, Telefon, Geburtsdatum, Kunden_seit, Letzter_Kontakt, Status
4
Make.com Szenario einrichten
make.com → Kostenlos registrieren → Neues Szenario → Webhook-Modul → URL kopieren
Diese URL als SHEETS_WEBHOOK_URL in Cloudflare eintragen → Google Sheets verbinden
5
Formspree Webhook aktivieren
formspree.io → Formular öffnen → Integrations → Webhook → Cloudflare Worker URL eintragen
Formular-ID: mlgpooqw — Test-Formular ausfüllen und prüfen ob E-Mail ankommt
6
Bestehende Kunden importieren
Kundenliste aus Excel/Programm als CSV exportieren → in "Richter_Kunden" Sheet importieren
Status für alle auf "Aktiv" setzen → System startet automatisch mit Jahrescheck + Geburtstagen
